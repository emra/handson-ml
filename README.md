## Useful commands

```bash
# create virtual env
pyenv virtualenv 3.9.12 myenv       

# activate virtual env
pyenv activate myenv  

# install packages
pip install tensorflow numpy pandas scikit-learn matplotlib torch transformers

# launch jupyter notebook
jupyter notebook 
```
